# Protocol Buffers

## Installation

### Homebrew

```sh
brew install protobuf
```

### APT

```sh
sudo apt update
sudo apt -y install protobuf-compiler
```

### YUM

```sh
sudo yum check-update
sudo yum -y install protobuf-compiler
```

### APK

```sh
sudo apk update
sudo apk add protobuf
```

## Protocol Compiler

```sh
go get -u github.com/golang/protobuf/{proto,protoc-gen-go}
```
