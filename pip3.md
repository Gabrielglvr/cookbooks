# Python Package Index (PIP3)

## Installation

### YUM

```sh
sudo yum check-update
sudo yum -y install epel-release
sudo yum -y install python36-pip
```

### APT

```sh
sudo apt update
sudo apt -y install python3-pip
```

### APK

```sh
sudo apk update
sudo apk add py3-setuptools
```

## Commands

```sh
pip3 -h
```
