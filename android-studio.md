# Android Studio

## Installation

### Homebrew

```sh
brew cask install android-studio
```

### Chocolatey

```sh
choco install -y androidstudio
```

## Uninstall

### Homebrew

```sh
brew cask uninstall android-studio
```

```sh
rm -fR ~/Library/Android
rm -fR ~/Library/Application\ Support/AndroidStudio3.4
```
