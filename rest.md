## Visual Studio Code

```sh
cat << EOF > ./.rest
@hostname = 127.0.0.1
@port = 8000
@host = {{hostname}}:{{port}}
@baseUrl = http://{{host}}/api/v1
@contentType = application/json

### Health check

GET {{baseUrl}}/health-check HTTP/1.1
Content-Type: {{contentType}}
EOF
```

```sh
code --install-extension humao.rest-client
```
