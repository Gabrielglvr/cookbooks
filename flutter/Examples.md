# Examples

## Doctor

```sh
flutter doctor -v
```

### Licenses

```sh
flutter doctor --android-licenses
```

## Emulators

```sh
flutter emulators
```

```sh
flutter emulators --launch [emulator]
```

```sh
flutter emulators --create --name [name]
```

## Config

```sh
flutter config --no-analytics
```

## Create

```sh
flutter create [name]
```

## Packages

```sh
flutter packages get
```

### Update

```sh
flutter update-packages
```

## Run

```sh
flutter run
```
