# Yarn

## Installation

### Homebrew

```sh
brew install yarn
```

### APT

```sh
sudo apt update
sudo apt -y install yarn
```

### YUM

```sh
curl -Ls https://dl.yarnpkg.com/rpm/yarn.repo | sudo tee /etc/yum.repos.d/yarn.repo
```

```sh
sudo yum check-update
sudo yum -y install yarn
```

### Chocolatey

```sh
choco install -y yarn
```

## Commands

```sh
yarn -h
```
