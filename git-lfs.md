# Git Large File Storage (LFS)

## Installation

### Homebrew

```sh
brew install git-lfs
```

### YUM

```sh
sudo yum check-update
sudo yum -y install epel-release
sudo yum -y install git-lfs
```

### APT

```sh
sudo apt update
sudo apt -y install git-lfs
```

### APK

```sh
sudo apk update
sudo apk add git-lfs
```

## Commands

```sh
man git-lfs
```
