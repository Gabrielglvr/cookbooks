# iputils

## Installation

### APT

```sh
sudo apt update
sudo apt -y install iputils-arping iputils-clockdiff iputils-ping iputils-tracepath
```

### YUM

```sh
sudo yum check-update
sudo yum -y install iputils
```

### APK

```sh
sudo apk update
sudo apk add iputils
```

## Commands

```sh
arping
clockdiff
ping
rdisc
tracepath
```
