# Python Package Index (PIP2)

## Installation

### YUM

```sh
sudo yum check-update
sudo yum -y install epel-release
sudo yum -y install python2-pip
```

### APT

```sh
sudo apt update
sudo apt -y install python-pip
```

### APK

```sh
sudo apk update
sudo apk add py2-pip
```

## Commands

```sh
pip2 -h
```
