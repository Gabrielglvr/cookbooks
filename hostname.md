# Hostname

## CLI

### Commands

```sh
hostname --help
```

### Tips

#### Show

```sh
sudo cat /etc/hostname
```

### Usage

#### Get IP

```sh
hostname -I
```

#### Set

```sh
sudo hostnamectl set-hostname [name] # ally, apex, axis, axel
```

```sh
hostnamectl status
```

```sh
sudo shutdown -r now
```
