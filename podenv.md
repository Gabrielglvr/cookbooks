# Pod Version Manager

## CLI

### References

- [Git Repository](https://github.com/kylef-archive/podenv)

### Installation

#### Homebrew

```sh
brew install kylef/formulae/podenv
```

### Commands

```sh
podenv -h
```

### Usage

```sh
# List Versions
podenv install --list

# Install
podenv install [version]

# Current version
podenv version

# Installed Versions
podenv versions

# Set Global
podenv global [version]

# Uninstall
podenv uninstall [version]
```
