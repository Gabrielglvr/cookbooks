# htop

## Installation

### Homebrew

```sh
brew install htop
```

### YUM

```sh
sudo yum check-update
sudo yum -y install epel-release
sudo yum -y install htop
```

### APT

```sh
sudo apt update
sudo apt -y install htop
```

### APK

```sh
sudo apk update
sudo apk add htop
```

## Commands

```sh
htop -h
```
