# Perl Compatible Regular Expressions (PCRE)

## Library

### Installation

#### APT

```sh
sudo apt update
sudo apt -y install libpcre3 libpcre3-dev
```

#### YUM

```sh
yum check update
# sudo yum -y install pcre pcre-devel
sudo yum -y install pcre pcre-devel
```

#### APK

```sh
apk update
apk add pcre pcre-dev
# apk add libpcrecpp
```
