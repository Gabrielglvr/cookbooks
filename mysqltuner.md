# MySQLTuner

## CLI

### References

- [README](https://github.com/major/MySQLTuner-perl#readme)

### Installation

#### Homebrew

```sh
brew install mysqltuner
```

#### APT

```sh
sudo apt update
sudo apt -y install mysqltuner
```

### Commands

```sh
mysqltuner --help
```

### Examples

#### Simple

```sh
mysqltuner \
  --host 127.0.0.1 \
  --user root \
  --pass root
```
