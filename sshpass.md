# sshpass

## CLI

### Installation

#### Homebrew

```sh
brew install http://git.io/sshpass.rb
```

#### APT

```sh
sudo apt update
sudo apt -y install sshpass
```

#### YUM

```sh
yum check-update
sudo yum -y install sshpass
```

### Commands

```sh
sshpass -h
```

### Usage

```sh
sshpass -p '[password]' ssh [username]@[hostname]
```
