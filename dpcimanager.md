## CLI

### Installation

#### Darwin Binary

```sh
curl \
  -L 'https://github.com/MuntashirAkon/DPCIManager/releases/download/2.0/dspci' \
  -o /usr/local/bin/dspci
```

```sh
chmod +x /usr/local/bin/dspci
```

### Commands

```sh
dspci -h
```
