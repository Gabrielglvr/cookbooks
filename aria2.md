# aria2

## CLI

### Installation

#### Homebrew

```sh
brew install aria2
```

#### APT

```sh
sudo apt update
sudo apt -y install aria2
```

#### YUM

```sh
yum check-update

# Repo: EPEL
sudo yum -y install aria2
```

#### Zypper

```sh
sudo zypper refresh
sudo zypper install -y aria2
```
