# Commands

## Connect

```sh
mongo
```

### Remote

```sh
mongo --host [hostname] --port 27017
```

### Authentication

```sh
mongo --host [hostname] --port 27017 -u [username] -p [password]
```

#### Database

```sh
mongo --host [hostname] --port 27017 -u [username] -p [password] --authenticationDatabase [db-name]
```
