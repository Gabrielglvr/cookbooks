## CLI

### Installation

#### Homebrew

```sh
brew install typescript
```

#### NPM

```sh
npm install typescript -g
```

### Commands

```sh
tsc -h
```

### Usage

```sh
# Initialize
tsc --init
```

### Tips

#### EditorConfig

```sh
cat << EOF > ./.editorconfig
[*.{ts,tsx}]
indent_size = 4
EOF
```
